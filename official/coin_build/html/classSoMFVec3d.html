Detailed Description

The SoMFVec3d class is a container for SbVec3d vectors.

This field is used where nodes, engines or other field containers needs to store an array of vectors with three elements.

See also
SbVec3d, SoSFVec3d, SoMFVec3f

Set num vector array elements from the xyz array, reading from index 0 in the given double-triplet vector array and storing values from index start in this SoMFVec3d's array.

If start + num is larger than the current number of elements in the SoMFVec3d field, this method will automatically expand the field to accommodate all given double-vector element values.
